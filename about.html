FAQ


What is a PWA?

To put it simply, a PWA is just like a website but has some special extra features which allow it to work better. You can read more about them here.


How do I install this site as a PWA?

It's actually very simple, though depends on your device and browser. Click here for some instructions on how to install a PWA. If you're on Android you can also install from the Play Store (currently in review).


Why did you build this?

I noticed that Amtrak's official tracking tool was way too data heavy to be used on any route with only 3G or even 2G. This led me down a path of making my own Amtrak Library for Node.js and then an api to go with it. I expected these to enable other develop tools similar to this, but I never saw any pop up. I then decided to take it upon myself to develop this site ad app, which is why you're here.


Can I see the source code and contribute?
